Week 1: Overview

The class begins by introducing the seven-layer OSI model, providing an overview that will serve as the foundation for a deeper exploration in later lessons.

Group # Layer Name Description
Program 7 Application Interacts with software and provides network services like email and web browsing.
6 Presentation Translates and formats data so the application can understand it.
5 Session Keeps track of who you are communicating with.
Operating System 4 Transport Ensures reliable data transfer between devices and controls how data is split into smaller chunks.
3 Network Handles routing of data across different networks and assigns logical addresses.
Hardware 2 Data Link Handles routing of data across different networks and assigns logical addresses.
1 Physical Deals with the physical connection between devices (copper wires, fiber optics, radio waves).
