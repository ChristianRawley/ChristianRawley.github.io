Week 1: Notation

The class begins by learning set-roster notation and set-builder notation. Below is a reference table for some symbols that will be used.

Symbol Meaning
ℝ All real numbers
ℤ All integer numbers
ℚ Set of all rational numbers
∈ Element of
∉ Not an element of
∣ Such that

A set is a collection of elements. The elements within a set can be anything - numbers, variables, or even other sets. We typically denote sets with a capital letter, and list out elements within curly braces to show that it is a set. When we do this, it is called set-roster notation.

Example: Let A = {1, 1, 4, 2, 7, 9}

If we want to say something is an element of a set, say 4, we can write: 4 ∈ A. This means '4 is an element of the set A', and conversely, 8 ∉ B means '8 is not an element of the set B'.

The axiom of extension says that sets are completely determined by its elements. This means the order of the set and number of times an element appears do not matter.
