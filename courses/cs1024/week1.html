Week 1: Introduction

The class begins with a simple pre-test to gauge what we know from previous courses.

Write a Java program named Prog.java that reads a string entered by the user and prints each word in the string on a new line. A "word" is defined as a sequence of non-space characters. The program should process the string character by character (Do not use the String split() or tokenizer() methods) and handle any number of spaces between words (Don't print blank lines).
